# PLC

Repository for SUTD Term 6 50.051 Programming Language Concept (PLC) 2026

## Build

To compile the code manually: 

gcc -Wall -Wextra -std=c11 main.c map.c enemy.c config.c save.c -o abyss_walker.exe

Otherwise:

make

## Run

Run in the terminal:

./abyss_walker.exe

## Clean

To remove the built executable:

make clean
